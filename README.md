User Management System using Python and MySQL

A command-line User Management System developed using Python and MySQL. The application allows users to register, log in securely, and manage their personal profile information.

After successful authentication, users can add and view profile details such as name, surname, email address, mobile number, and age. All user credentials and profile information are stored in a MySQL database.

The project demonstrates the use of Object-Oriented Programming (OOP), MySQL database connectivity, SQL queries, exception handling, and user-specific data management in Python.
